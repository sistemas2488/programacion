Ejemplo ciclo Repetir

Ejemplo

Escribir un programa que solicite la carga de N números entre 1 y 999 para obtener la siguiente relación: números que tienen un dígito, números que tienen dos dígitos y números que tienen tres dígitos. Por cada número el programa debe decir al instante a que categoría pertenece. El programa finalizar cuando se cargue o presione el valor 0 y a continuación debe indicar al final cuantos números se almacenaron para cada categoría.
