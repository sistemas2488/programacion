Entrada y salida de datos

Entrada de datos

Para la entrada de datos se tiene diferentes herramientas para la entrada de datos como:

  • Scanner
  • JoptionPane
  • BufferReader
  • Archivos de Texto

Para esta unidad se aplicara la clase Scanner.

Clase Scanner: Scanner es una clase en el paquete java.util utilizada para obtener la entrada de los tipos primitivos como int, double etc. y también String. Es la forma más fácil de leer datos en un programa Java, aunque no es muy eficiente si se quiere un método de entrada para escenarios donde el tiempo es una restricción, como en la programación competitiva. Para utilizar la clase es necesario crear un objeto de dicha clase

Salida de datos

Para la entrada de datos se tiene diferentes herramientas para la entrada de datos como:

  • System.out.print
  • JoptionPane
  • Archivos de Texto

Para esta unidad se utilizara System.out.print
